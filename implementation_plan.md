# Bavozan Group WHOOP Website Plan

## Files:
- index.html - Main storefront
- admin.html - Admin panel (password protected)
- styles.css - Dark burgundy/black theme
- app.js - Main logic, orders, delivery
- admin.js - Admin panel logic
- logo.png - Already provided by user

## Features:
- Multi-language: Badini, Sorani, Arabic (RTL)
- Products: WHOOP 4.0 Peak, WHOOP 5.0 (all gens), Bands, Used with/without subscription
- Order system with city-based delivery pricing
- Kurdistan cities: 3,500 IQ
- Other Iraq cities: 5,000 IQ
- Admin: Add/view/manage orders and listings with photo upload
- Social links: TikTok, Instagram, WhatsApp, Phone
- Location: Erbil, Started 2026
